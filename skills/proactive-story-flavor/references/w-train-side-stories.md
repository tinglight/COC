# W列车支线故事风味

当用户要求 W列车，或类似火车/车站/公司交通模组的主动故事风味、支线小故事、伏笔揭露、填充灵感时，使用这份参考资料。

## 玩家安全边界

把 W列车风味当成车站周边的公开压力和普通社会后果来写。使用售票处、失物处、投诉、维修、小报、寻人启事、员工规则和乘客日常。只揭示支线局部回收，例如规则改变、被隐藏的投诉、归还的物件或矛盾的公开记录。

不要揭露真实旅行时长、运输机制、公司秘密、隐藏时间线、必需线索链、结局、犯人逻辑或 PC 的未来。如果一个电影化恐怖场景会暴露这些内容，就把它转成公开症状：票上的错误日期、锁住的柜子、员工沉默，或没人愿意签名的记录。

## 核心调色盘

公开压力：
- 准点考核、投诉压制、家属寻人启事、退票纠纷、员工保密规则、深夜车站宵禁、行李遗失索赔、职场追责。

普通人物：
- 售票员、检票员、失物处职员、清洁工、站台巡逻员、小报编辑、排字工、维修工、乘客家属、茶水间服务员、站务主管。

痕迹物件：
- 旧票根、黑色封箱、错日期印章、失物账本、热水壶、磨损一半的铅字、维修报告、手写告示、窗上掌印、湿漉漉的站台硬币、退票单。

安全扰动：
- 明天的车票显示已使用、空座被预留、行李箱自己换柜格、投诉里少了两个字、水壶一直是温的、维修线从站台移到车门、告示照片变化、广播磁带使用某个员工的声音。

支线回收：
- 本地规则新增、员工拒绝签字、物件被锁起来、报告被归到错误日期、家属改变说法、公开告示被重写、退票理由变得无法解释。

## 小弧模板

一个支线故事用 3-4 次播报展开：

```text
起: [车站日常] 在 [痕迹物件] 上显示 [错误细节]。
承: [普通人/机构] 在 [公开压力] 下回应，造成 [代价或沉默]。
转: [目击者/物件/记录] 把错误细节连到早先痕迹，但不解释主因。
收: [局部回收] 改变车站日常，并留下小钩子。
```

成品播报应保持 1-2 句中文短句。规划时可以显示节拍标签；运行时播报输出则省略标签，除非用户要求规划材料。

## 可用支线链

### 失物处的黑箱

- 起: 车站失物处多了一只没有编号的黑色行李箱，箱牌上写着“十秒后领取”。
- 承: 夜班职员发现箱子每天都会换到更靠外的柜格，像是有人从里面慢慢把它推出来。
- 转: 一名乘客家属来认领时，报出的不是箱子外观，而是“里面敲窗的节奏”。
- 收: 职员最后在箱底找到一张旧退票单，退票理由只有四个字：本人未归。

### 盖错日期的车票

- 起: 售票窗口今天连续打出三张“明天已使用”的车票，售票员只好把它们夹进作废票夹。
- 承: 晚班交接时，作废票夹里少了一张，却多出一枚湿漉漉的站台硬币。
- 转: 清洁工说，他在月台边见过有人拿着同样的票，问“这一次能不能少等几年”。
- 收: 那枚硬币后来被发现卡在检票机内部，正好挡住了“已进站”的字样。

### 第三候车室的热水

- 起: 第三候车室的茶水间每晚都会多烧一壶热水，水壶旁压着一张三等座车票。
- 承: 值班员以为是乘客遗落，直到他发现车票日期每天都比当天早十秒。
- 转: 有位老人坚持坐在水壶旁等人，说对方每次回来都只来得及喝一口。
- 收: 车站后来撤掉了那只水壶，但墙上仍留下一个圆形水印，摸上去总是温的。

### 被删掉的报纸短讯

- 起: 地方小报原本要刊登一则 W 列车投诉，临印前却删掉了“十秒”两个字。
- 承: 排字工发现那两个铅字被磨得只剩半边，像是已经被反复使用了很多年。
- 转: 编辑收到匿名信，信里只要求他们以后不要把“准点”写成“归来”。
- 收: 第二天报纸照常发行，只是在车站广告栏里，那条空白短讯被人用铅笔补上了乘客姓名。

### 永远空着的座位

- 起: 售票系统总会自动保留一张靠窗座，备注栏写着“请勿安排活人”。
- 承: 新来的售票员删掉备注后，那趟车所有靠窗票都被退回，退票时间完全相同。
- 转: 站务主管把她叫去谈话，只问了一句：“你看见座位上的人了吗？”
- 收: 从那天起，售票窗口旁多了一张手写告示：靠窗座售出前，请先确认玻璃里只有一个倒影。

### 广播室的备用磁带

- 起: 广播室发现一盘没有标签的备用磁带，播放时只有很轻的呼吸声。
- 承: 技术员准备销毁它，却听见磁带里有人用他的声音念出了下一班列车时刻。
- 转: 广播员说这不是第一次，上一位技术员也是从磁带里听见自己辞职的。
- 收: 磁带最后被锁进铁柜，柜门外贴着检修单：如听见本人声音，请不要回答。

### 维修工的裂缝记录

- 起: 站台维修工在同一块地砖上连续三天记录裂缝，可每次裂缝形状都不一样。
- 承: 他用粉笔描边，第二天粉笔线却出现在车厢门内侧。
- 转: 工长要求他别再写报告，因为上一份报告的签名人已经“提前离职”。
- 收: 维修工偷偷把裂缝拓印下来，发现那不是裂纹，而像一排紧贴玻璃的手指。

### 家属寻人栏

- 起: 车站公告栏贴满寻人启事，其中一张每天都会自己换成新的照片。
- 承: 照片上的人越来越年轻，最后变成一个还没买票的学生。
- 转: 学生的母亲来撕启事，却说“这孩子明明还在家里睡觉”。
- 收: 启事背面留下售票窗口的排队号，而那个号码第二天正好叫到了学生本人。

## 填充提示

生成新的 W列车支线故事时，询问或推断这些字段：

```text
支线物件: [车票 / 行李箱 / 广播磁带 / 检修单 / 热水壶 / 退票章 / 寻人启事]
普通角色: [售票员 / 清洁工 / 家属 / 小报编辑 / 维修工 / 巡警 / 夜班站务]
第一伏笔: [日期不对 / 声音不对 / 数量不对 / 影子不对 / 记录被改]
揭露方式: [账本对上 / 物件回到原处 / 证人改口 / 旧记录出现 / 规则被迫写成告示]
收尾余味: [规定改变 / 某人沉默 / 一个物件被锁起 / 日常继续但多了禁忌]
```

可复用句式：

```text
因为[公共压力]，[普通角色]选择[小异常处理方式]，留下[具体痕迹]，也让[后续疑问]变得不好解释。
```
